# Waveshare 2.4" LCD Setup

Simple "Hello World" for the Waveshare 2.4-inch SPI LCD on Raspberry Pi.

## Installation

1. **Run the install script:**
   ```bash
   chmod +x install.sh
   ./install.sh
   ```
   *Note: If SPI was just enabled, you must reboot your Pi.*

2. **Run the Hello World example:**
   ```bash
   source .venv/bin/activate
   python hello_world.py
   ```

## Requirements
- Raspberry Pi with SPI enabled.
- Waveshare 2.4inch LCD Module.
